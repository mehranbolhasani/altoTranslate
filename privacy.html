Privacy — altoTranslate

This page describes how the altoTranslate Chrome extension handles your information. It is written to be clear and short, not as legal advice.

What the extension can see

  • Selected text on web pages when you intentionally use the extension (for example after you highlight text and open the translation popup or use the keyboard shortcut). The extension only processes text you select in that way.
  • Your settings stored in the browser, such as target language, theme, and optional altoCloud API key.
  • Saved vocabulary words you choose to keep, stored locally in your browser.

Where translation requests go

altoTranslate translates text through our own translation service at api.altotranslate.xyz (operated by altoTranslate / altotranslate.xyz).

  • Free tier — no API key required. Translations are rate-limited (currently 100 per day). Your browser sends the selected text and language settings to our API to produce a translation.
  • altoCloud Pro — requires an API key from altotranslate.xyz/dashboard. The key is stored in Chrome extension storage on your device and sent with each translation request. Pro has unlimited translations.

We do not route translations through third-party APIs you configure yourself (such as Gemini, DeepL, or Azure). Those providers are no longer used by the extension.

What we store

  • On your device: extension settings, optional altoCloud API key, saved vocabulary, translation cache, and usage counters (for example daily free-tier limits and review prompts).
  • On our servers: when you translate, we receive the text you selected and the language parameters needed to return a translation. We use this only to provide the translation service and enforce free-tier limits or validate Pro keys. We do not sell your data.

Cached translations

To speed up repeat lookups, the extension may cache some successful translation results locally in your browser. You can clear the translation cache from the extension settings.

Contact

For questions about this extension or privacy, use the support or repository links from the Chrome Web Store listing or visit altotranslate.xyz.
